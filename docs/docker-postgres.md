# הרצת PostgreSQL באמצעות Docker (Development)

מסמך זה מסביר איך להריץ PostgreSQL באמצעות Docker עבור פרויקט BuyForce.
השימוש בדוקר הוא אופציונלי אך מומלץ לצוות.

---

## דרישה מקדימה

- Docker Desktop מותקן ורץ על המחשב
- אין צורך בהתקנת PostgreSQL מקומית

---

## שלב 1 – הרצת PostgreSQL

### Windows (PowerShell / CMD)

```bash
docker run --name buyforce-postgres ^
  -e POSTGRES_USER=postgres ^
  -e POSTGRES_PASSWORD=123456 ^
  -e POSTGRES_DB=buyforce ^
  -p 5432:5432 ^
  -d postgres:16

 


#  ....................................................................................................
#     שלב 2 בדיקה שה-DB רץ    

docker ps

# אמור להופיע container בשם:

buyforce-postgres

# .....................................................................................................
# שלב 3 הגדרת BACKEND

בקובץ nest-api/.env:

DB_HOST=localhost
DB_PORT=5432
DB_USER=postgres
DB_PASSWORD=123456
DB_DATABASE=buyforce
 

#  אין לשנות ערכים אלה כאשר משתמשים ב־Docker!!!!.


# ......................................................................................................
# שלב 4 הרצת ה-BACKEND 

cd nest-api
npm run start:dev

# אם אין שגיאות חיבור – PostgreSQL מחובר בהצלחה.


# .......................................................................................................
# .......................................................................................................
# פקודות שימושיות 


# עצירת ה-DB

docker stop buyforce-postgres


#  הפעלה מחדש

docker start buyforce-postgres


# מחיקה והתחלה נקייה:

docker rm -f buyforce-postgres
# לאחר מחיקה יש להתחיל שוב את שלב 1
